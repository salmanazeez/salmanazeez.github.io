Architecture website

I was inpsired by an UI design on Behance by Andrey Lohmatov which he has designed for an architecture company. I was impatient to bring the desgin into life in the browser. I started building the web app using React.

On the go, realized that web app was simple yet significant with some cool features. I used Material UI along with the web app. Web app is fully responsive and cross-browser compatible. Homepage has a cool and simple background slider (carousel) and I used React Background Slider package.

I was a beginner using React. Building this template helped me to build myself and gain a thorough knowledge of React and Material UI.

Clone the repo or view the code: github.com/salmanazeez/architecture-app

GitHub repo: architecture-app

Technologies and tools

React, Material UI, Sass, React Bootstrap, GitHub, VS Code
